FossXO Game Design Document¶

Welcome to FossXO game design document. FossXO is a free and open-source example of tic-tac-toe created using the Rust programming language. This document describes in detail the design considerations of the game.

The FossXO game design document is also available in these other formats:

  • Printable PDF

  • Single HTML page
